4X Space Strategy Game Roadmap (Updated July 2026)
Core Foundation (Current Progress – Very Solid)

Solar System Generation
Random star + planets with varied types (Gas Giant, Volcanic, Ice, Ocean, Rocky, Barren, etc.)
Realistic-ish orbital mechanics with speed scaling (Kepler-like)
Moons with their own orbits around planets
Visual blue orbit rings that follow hosts
Clickable bodies with Planet UI + surface grid

Visualization & Camera
3D system view with colored bodies
Smooth camera panning (WASD) + elevation change (mouse wheel)
Basic raycast selection + debug tools

UI System
Planet info panel (bottom left)
Draggable/resizable Planet Grid Window with rectangular surface
Sandbox Editor Panel (size, radius, speed sliders, Apply Changes)
Terrain Editor Panel for testing biome generation


Major Features In Progress / Recently Added

Sandbox Mode (High Priority – Almost Done)
Real-time editing of selected body (planet/moon)
Radius slider (moons now jump + rings should resize)
Size slider (stable scaling)
Speed slider
Parent/child relationships preserved for moons
Debug logging for tracing issues

Planet Surface System
Rectangular grids (map-like view)
Improved noise-based biome generation (mountains, volcanoes, magma fields, etc.)
Seamless horizontal scrolling (in progress)
Terrain Editor with sliders for regeneration testing
